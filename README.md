## 💻 Déploiement

Optionnel : Renommer .env.exemple en entrant des valeurs aux variables
```
MYSQL_DATABASE:
MYSQL_USER:
MYSQL_PASSWORD:
MYSQL_ROOT_PASSWORD:
```

Lancer les services Docker

```
docker compose up
```
